# Shopping App

## Dev-apps

1. go to dev-apps folder
2. run `docker-compose up`

## Api

1. go to `api` folder
2. run `npm run install-globals` ( in case you dont have typescript/nodemon)
3. run `npm install`
4. run `npm start`
